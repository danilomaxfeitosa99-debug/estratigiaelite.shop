# AeroJet Executive Aviation - Versão CNPJ 60.862.894/0001-50

Esta pasta agrupa uma variação do site institucional da **AeroJet Executive Aviation**, configurada para uso com o CNPJ **60.862.894/0001-50** e com textos ajustados em relação à versão original.

## Visão Geral

O site apresenta a frota, os serviços de fretamento e gestão de aeronaves, além de canais de contato voltados a operações 24/7 e atendimento a clientes corporativos e particulares.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Luxuoso, responsivo e otimizado para campanhas digitais.
-   **Público-alvo:** Executivos, famílias e empresas que utilizam aviação executiva com frequência.
